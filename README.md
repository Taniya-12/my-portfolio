# 👋 Taniya Kumari — Developer Portfolio

A modern, animated developer portfolio built using **React 19**, **TypeScript**, **Vite**, **Tailwind CSS v4**, and **Framer Motion**.

---

## 🚀 Tech Stack

- ⚛️ React 19
- 🟦 TypeScript
- ⚡ Vite
- 🎨 Tailwind CSS v4
- 🎭 Framer Motion
- 🌀 Lenis (Smooth Scroll)
- ⌨️ React Type Animation
- 🔗 React Scroll
- 🎯 Lucide React
- React Icons

---

## ✨ Features

- Responsive Design
- Smooth Scrolling
- Animated Sections
- Sticky Sidebar
- Interactive Navigation
- Project Showcase
- Skills Progress Bars
- Education Timeline
- Experience Timeline
- Contact Form
- Glassmorphism UI
- Accessible Animations

## ⚙️ Installation

```bash
git clone <your-repo-url>

cd my-portfolio

npm install

npm run dev
```

---

## 📦 Build

```bash
npm run build
```

Preview production build

```bash
npm run preview
```

---

## 🎨 Customize

Update these files to personalize the portfolio.

```
src/data/
├── personal.ts
├── education.ts
├── experience.ts
├── projects.ts
├── skills.ts
└── navigation.ts
```

Place your assets inside

```
public/
├── profile.jpg
├── resume.pdf
└── projects/
```
